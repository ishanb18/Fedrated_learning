# Model Checkpoints

Place trained model checkpoints (`.pth` / `.pt`) here so they are tracked on GitHub.

- **Naming**: e.g. `fl_ucmerced_final.pth`, `fl_nwpu_resisc45.pth`
- **Size**: GitHub allows files up to 100 MB. For larger models, use [Git LFS](https://git-lfs.github.com/) or host elsewhere and link in the main README.

Checkpoints in this folder are **not** ignored by git and will be pushed to the repo.
