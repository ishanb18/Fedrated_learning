# Report & Results

Place your project report and result artifacts here so they are tracked on GitHub.

- **Report**: Add your written report as a PDF (e.g. `report.pdf`) or use `RESULTS.md` for a markdown summary.
- **Results**: Add screenshots, plots, or tables (e.g. `.png` images) that summarize accuracy, F1, AUC, or communication cost.

These files are not ignored by git and will be pushed to the repo.
